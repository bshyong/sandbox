# Reactagram

## Purpose

An implementation of Instagram for practice using React to build an isomorphic Javascript app.

## Success Criteria

All UI components should be constructed using React. Application views should be rendered server-side in an isomorphic fashion.

## Architecture overview
- Node.js server
  - Passport for auth
  - React for views
- Jasmine for tests
